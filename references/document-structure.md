# 学位论文组成部分与顺序(2025版)

**适用学位:博士与硕士通用。** 博硕的组成部分、顺序、页码与篇眉规则完全相同;仅封面的学位类别行(博士/硕士、学术/专业)与「关于学位论文使用授权的说明」所选版本(博士版 03-1 / 硕士版 03-2)按学位类型不同——学位类型由 `scripts/detect_degree.py` 从封面/英文封面自动识别。

| 序 | 部分 | 必备 | 页码 | 篇眉 | 备注 |
|---|---|---|---|---|---|
| 1 | 中文封面 | ✓ | 无 | 无 | 学校模板,单面印刷,特殊页边距;学位类别行标明博士/硕士、学术/专业(学位识别依据) |
| 2 | 英文封面 | ✓ | 无 | 无 | 同上;学位行 Doctor of Philosophy / Master of …(学位识别依据) |
| 3 | 指导小组、公开评阅人和答辩委员会名单 | ✓ | 无 | 无 | 单面印刷 |
| 4 | 关于学位论文使用授权的说明 | ✓ | 无 | 无 | 单面印刷,签名页;博士版/硕士版二选一,标题相同 |
| 5 | 中文摘要 | ✓ | Ⅰ 起(罗马) | 摘要 | 双面印刷自此开始 |
| 6 | Abstract | ✓ | 罗马续 | Abstract | |
| 7 | 目录 | ✓ | 罗马续 | 目录 | 列至二级节 |
| 8 | 插图和附表清单 | 有图表则须 | 罗马续 | 同标题 | 可拆"插图清单"+"附表清单" |
| 9 | 符号和缩略语说明 | 可选 | 罗马续 | 同标题 | 符号少可不设 |
| 10 | 正文 | ✓ | 阿拉伯 1 起 | 各章标题 | 从另页右页(奇数页)开始 |
| 11 | 参考文献 | ✓ | 阿拉伯续 | 参考文献 | 另起页 |
| 12 | 附录 | 可选 | 阿拉伯续 | 附录X 标题 | 编号 附录A、附录B…… |
| 13 | 致谢 | ✓ | 阿拉伯续 | 致谢 | 限一页 |
| 14 | 声明 | ✓ | 阿拉伯续 | 声明 | 单独一页,统一文本 |
| 15 | 个人简历、在学期间完成的相关学术成果 | ✓ | 阿拉伯续 | 同标题 | 无成果写"无";子标题+类别标签+成果条目混排,整段保留 |
| 16 | 指导教师(小组)评语 | ✓ | 阿拉伯续 | 同标题 | 与审批材料一致 |
| 17 | 答辩委员会决议书 | ✓ | 阿拉伯续 | 同标题 | 与审批材料一致 |
| 18 | 其他材料 | 可选 | 阿拉伯续 | — | 置于最末 |

附录内编号规则:图 A.1、表 B.2、式(C-3)——冠以附录序号,与正文分开编。

页码均位于页脚居中,Times New Roman 五号,无修饰线。篇眉五号宋体(英文数字 TNR)居中,奇偶页相同,各部分首页也有。

> **本 skill 对模板专用版式的处理**:中文/英文封面、学位论文指导小组与答辩委员会名单、符号和缩略语说明、个人简历及学术成果等部分,用学校模板排好的专用版式(封面专用页边距、信息表、tab 两列对齐、居中子标题、参考文献式条目等),整改时**整段保留不动**,只对摘要起的正文部分按规范整改。
